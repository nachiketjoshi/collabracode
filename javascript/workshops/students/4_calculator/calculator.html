To make this work, you must implement the following in student_calculator.js:
  • button handlers
  • memory value
  • current value as a string
  • C = remove current operation, set current value to 0, and display 0
  • MC = Memory Clear
  • M+ = Add to existing value in memory and display result
  • M- = Subtract from existing value in memory and display result
  • MR = Show current memory value
  • + = Add
  • - = Subtract
  • * = Multiply
  • / = Divide
  • +-/ = Negate
  • Allow up to one "." in the string
