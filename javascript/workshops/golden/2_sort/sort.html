Testing

If on Linux or Mac, go to git base directory and run python -m SimpleHTTPServer.
Update sort_test.html by filling in the test cases.
Go to http://localhost:8000/jsunit/testRunner.html
Run the test at http://localhost:8000/tutorials/students/2_sort/sort_test.html
